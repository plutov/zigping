# zigping

Inspired by [gping](https://github.com/orf/gping).

## Usage

```bash
zigping <hostname1> <hostname2> ...
```

## Resources

- https://github.com/rockorager/libvaxis
- https://github.com/orf/gping
- https://github.com/zigcc/awesome-zig
- https://zig.guide/

## Build

See `build.zig`.

```bash
zig build
```
